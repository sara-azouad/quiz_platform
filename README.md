# Quiz Platform 

## Description
Quiz Platform is a web-based quiz management application developed to provide an interactive and user-friendly environment for creating, managing, and taking quizzes online.

The platform allows users to authenticate securely, participate in quizzes, submit answers, and view their results and history. An administration section is also included for managing quizzes and users.

This project was developed to improve backend development skills, database management, authentication systems, and web application design using PHP technologies.

---

## Features
- User authentication system
- Online quiz participation
- View quiz results and history
- Password reset functionality
- Admin dashboard
- Responsive user interface
-  Dynamic quiz submission system
- Secure access management

## Installation

1. Clone the repository

```bash
git clone https://github.com/saraazouad/quiz_platform.git
```

2. Open the project folder

```bash
cd quiz_platform
```

3. Configure your database connection in `db.php`

4. Import the database into MySQL

5. Start your local server (XAMPP, WAMP, etc.)

6. Open the project in your browser
